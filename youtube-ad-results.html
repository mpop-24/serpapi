YouTube Ad Results API

When a YouTube search contains advertisements, they are parsed and exist within the ads_results array in the JSON output.

API Examples

Results for: video ads

When SerpApi encounters video ads results, we add them to our JSON output as the array ads_results. For each ads result, we are able to extract its title, link, channel, views, thumbnail and more.


JSON Example

{
  ...
  "ads_results": [
    {
      "position_on_page": 1,
      "title": "LEGO Star Wars Ahch-To Porg Pandemonium",
      "link": "https://www.youtube.com/watch?v=eV7aLkblQys",
      "serpapi_link": "https://serpapi.com/search.json?engine=youtube_video&v=eV7aLkblQys",
      "channel": {
        "name": "LEGO",
        "link": "https://www.youtube.com/user/LEGO"
      },
      "views": 1300000.0,
      "description": "Watch Luke and Rey get interrupted by a determined group of Porgs!",
      "length": "1:26",
      "thumbnail": {
        "static": "https://i4.ytimg.com/vi/eV7aLkblQys/0.jpg",
        "rich": "https://i.ytimg.com/an_webp/eV7aLkblQys/mqdefault_6s.webp?du=3000&sqp=CM6W9fwF&rs=AOn4CLCGK-XJOEnYo_iNGtkkbLNarYFLSA"
      }
    }
  ],
  ...
}
