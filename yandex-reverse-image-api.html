Yandex Reverse Image API

Yandex Reverse Image API allows you to get results for an image search. When we query Yandex Images API with the URL of the image, we can get results such as image_preview, image_results, image_sizes,image_sizes_message, shopping_results, image_tags, similar_images, and knowledge_graph. They are parsed and exist within the JSON output.

To scrape Yandex Reverse Image with SerpApi, create a search with these parameters:

API Parameters

url

Required

Parameter defines the URL for an image to perform the reverse image search.

crop

Optional

Parameter is used to crop the image and perform the reverse search on the cropped part of the image. E.g. 0.04;0.46;0.27;0.84.
These coordinates are formatted in the next order: left edge;top edge;right edge;bottom edge. The minimum and maximum for each coordinate are 0 and 1, respectively.

crop_id

Optional

Parameter is used to filter results by a specific section of an image. Parameter will only work with images uploaded by Yandex (e.g. https://avatars.mds.yandex.net/rest-of-the-image-url).

crop and crop_id parameters should not be used together.
