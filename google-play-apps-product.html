Google Play Apps Product API

The Google Play Product API allows a user to scrape the results of Google Play Product Apps store. SerpApi is able to make sense of this information and extract product_info, media, ratings, reviews, similar_results, and more.

API Parameters

store

Required

Parameter must be set to apps. (I.e., store=apps)

Example with product_id: com.netflix.mediaclient
