Google Trends Trending Now Daily API

The Google Trends Trending Now Daily API scrapes the daily search trends section of Google Trends Trending Now page. It scrapes query, traffic, explore link, related queries and articles. In addition, serpapi_pagination can be used to retrieve the next page results of daily search trends.

This layout is deprecated by Google. For the latest version, use Google Trends Trending Now API. To use this old layout, set frequency to daily.

API Parameters

frequency

Required

Set parameter to daily to retrieve daily search trends.

date

Optional

Parameter defines the date of the results you want to retrieve. It defaults to the current day. The date format is yyyymmdd (e.g., 20231015).

JSON structure overview 

{
  ...
  "daily_searches": [
    {
      "date": "String - Date of the searches result",
      "searches": [
        {
          "query": "String - Query of the search",
          "google_trends_link": "String - Link of the Google Trends explore page of the query",
          "serpapi_google_trends_link": "String - Link of Google Trends Trending Now results of the query in JSON by SerpApi",
          "traffic": "Integer - Total traffic of the query",
          "related_queries": "Array - Related queries which consists of query, google_trends_link and serpapi_google_trends_link",
          "articles": [
            {
              "title": "String - Title of the article",
              "link": "String - Direct link of the article",
              "snippet": "String - Snippet of the article",
              "source": "String - Source of the article",
              "date": "String - Date of the article published date",
              "thumbnail": "String - Thumbnail of the article"
            },
          ]
        }
      ]
    },
  ],
  "serpapi_pagination": {
    "current_date": "String - Current date (e.g. 20230717) of the search results",
    "next_date": "String - Next date (e.g. 20230716) of the search results",
    "next": "String - Link to the next date of Google Trends Trending Now results in JSON by SerpApi"
  }
}
