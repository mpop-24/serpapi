Ruby sample code using stick parameter

To get a knowledge graph carousel in a reliable way, you need to extract the stick parameter. The stick parameter is present on pages where a regular knowledge graph panel is displayed. It's usually inside the link "View x+ more". The link is extracted as {block}_link in your SerpApi JSON.

require 'http'
first_response = HTTP.get 'https://serpapi.com/search.json?q=McDonald%27s&location=Dallas&hl=en&gl=us&source=test'
first_json = first_response.parse
people_also_search_for_link = first_json["knowledge_graph"]["people_also_search_for_link"]
stick_parameter = CGI.parse(URI.parse(people_also_search_for_link).query)["stick"][0]
second_response = HTTP.get "https://serpapi.com/search.json?q=McDonald%27s&location=Dallas&hl=en&gl=us&source=test&stick=#{stick_parameter}"
second_json = second_response.parse
second_json["knowledge_graph"]["people_also_search_for"]
