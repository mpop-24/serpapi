Google Play Books Product API

The Google Play Product API allows a user to scrape the results of Google Play Product Audiobooks store. SerpApi is able to make sense of this information and extract product_info, ratings, reviews, about_the_author, similar_results, and more.

API Parameters

store

Required

Parameter must be set to books. (I.e., store=books)

API Examples

Example with product_id: aWZzLPhY4o0C
