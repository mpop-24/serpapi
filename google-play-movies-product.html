Google Play Movies Product API

The Google Play Product API allows a user to scrape the results of Google Play Product Audiobooks store. SerpApi is able to make sense of this information and extract product_info, media, cast_and_credits, ratings, reviews, similar_results, and more.

API Parameters

store

Required

Parameter must be set to movies. (I.e., store=movies)

API Examples

Example with product_id: ePpJDKfRAyM
